# **sfacg-backups**

## usage

- #### `login <account> <password>` login your account and save cookies 
- #### `download <bid> | <url>` download book text and images
- #### `search <keyword>` search books by keyword 

- #### Example
  - ``` download 100003327 ```
  - ``` login account password ```
  - ``` search keyword ```
